# 📊 Instant Data Intelligence

> Transform any CSV or Excel file into a full analytics dashboard in seconds. No code. No setup. Just upload and explore.

## What It Does

Instant Data Intelligence is a production-ready data analytics app built with Python + Streamlit. Drop in any spreadsheet and instantly get:

- Trend analysis — time series with automatic date detection
- Distribution charts — box plots and proportion breakdowns
- Correlation heatmaps — find hidden relationships in your data
- Auto insights — volatility, top categories, strongest correlations surfaced automatically
- Data quality report — nulls, duplicates, and outliers flagged instantly
- Export — download your cleaned data with one click

## Tech Stack

- Frontend: Streamlit + Custom CSS (glassmorphism UI)
- Data processing: Pandas + NumPy
- Visualization: Plotly Express + Plotly Graph Objects
- Column intelligence: Auto-classification (numeric, categorical, date, text, ID)

## Smart Features

### Auto Column Classification
The app automatically detects column types — no manual configuration needed.

### Instant Insights Engine
Every dataset surfaces 3 automatic insights:
1. Most volatile metric — highest variance column
2. Top category — which segment drives the most volume
3. Strongest correlation — the two variables most connected

### Data Quality Report
Before you even look at a chart, the app flags nulls, duplicates and outliers.

## Running Locally

1. git clone https://github.com/your-username/instant-data-intelligence
2. cd instant-data-intelligence
3. python -m venv venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. streamlit run app.py
